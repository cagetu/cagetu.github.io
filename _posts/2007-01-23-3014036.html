그 동안 정신없어서 못했던 작업들을 다시 시작..

-Scene(World) 하나를 n개의 Viewport 가 공유해서 그릴수도 있어야 한다.
-각각의 Viewport는 다른 Camera를 세팅할 수 있어야 한다.

*Window
: 실제 윈도우에 대한 정보를 가진다. n개의 viewport를 가질 수 있음.

*Viewport
: 랜더링을 될 영역 Camera를 하나 세팅할 수 있음.

*Camera
: 장면 하나를 세팅해서 보여준다.

*Scene
: 실제 랜더링 될 Scene 외부에서 다른 Viewport에 설정할 수 있어야 함.

*RenderTarget
: Window + D3D Device 정보를 가진다..

*Renderer
: p3D 객체를 가진다. n개의 RenderTarget을 가질 수 있다. 즉, 크게 보면, n개의 윈도우의 랜더링을 관리

*Main
: Scene(World), Camera 객체는 공유, 변경 될 수 있어야 함으로, 이 녀석들을 관리한다.

*Framework
: Entrypoint Class
--------------------------------------------------------------------------------
<psuedo>
Main* pMain = new Main;
Camera* pCam = pMain->CreateCamera( aa );
Scene* pScene = pMain->CreateScene( bb );

pCam->SetScene( pScene );

Window* pWindow = new Window( hWnd, 640, 480 );

Viewport* pViewport = pWindow->CreateViewport( 영역 );
pViewport->SetCamera( pCam );

// 카메라 변경시..
pCam = pMain->CreateCamera( cc );
pCam->SetScene( pScene );

pViewport->SetCamera( pCam );

// 장면 변경시
pScene = pMain->CreateScene( dd );
pCam->SetCamera( pScene );

--------------------------------------------------------------------------------
<업데이트는 부모/자식 관계로 인해>
Renderer->RenderTarget(n개)->Window->Viewport(n개)->Camera->Scene 순으로 업데이트 됨
