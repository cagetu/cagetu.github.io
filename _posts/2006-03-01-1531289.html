mirror 라는 것이 내부적으로는 node를 local 좌표계로 -스케일을 적용시킨 것이다..
Node.NoScale()을 하면, 내부적으로 mirrored 된 것을 다시 원상복구 시킨다.
스케일을 빼버리면, mirror한 효과를 얻을 수 없으므로, NoScale()을 적용하면, 아마 캐릭터가 접혀버릴 걸.. ㅎ

어차피 애니메이션에서는 rotate와 trans 값이 중요 하기 때문에,
mirror된 본 Matrix를 분해해서 Scale을 빼고, rotation, trans 값만을 다시 매트릭스로 만든어서 미러 된 본에 적용시키면, 미러된 상태에서의 rotate/trans를 적용시킬 수 있다.. 물론 본의 스케일을 적용시키려면, scale -> -scale로 해서 적용하면 될지도..(이건 해보지 않아서 정확하지 않을 듯)

자세한 내용은 밑에 예제 코드를 적용해 보도록 하자.

void Uniform_Matrix( Matrix3& In, Matrix3& Out )

 AffineParts parts;
 decomp_affine( In, &parts );

 parts.q.MakeMatrix( Out );

CODE FOR REMOVING NON_UNIFORMING SCALING FROM 3DSTUDIO MAX NODE TRANSFORMS (USEFUL FOR BIPED)

//UNIFORM_MATRIX REMOVES
//NON-UNIFORM SCALING FROM MATRIX

//must include this in your file's header #include "decomp.h"

Matrix3 Uniform_Matrix(Matrix3 orig_cur_mat) {

AffineParts parts;

Matrix3 mat;

///Remove scaling from orig_cur_mat

//1) Decompose original and get decomposition info

decomp_affine(orig_cur_mat, &parts);

//2) construct 3x3 rotation from quaternion parts.q

parts.q.MakeMatrix(mat);

//3) construct position row from translation parts.t

mat.SetRow(3, parts.t);

return(mat);

}

//GET_RELATIVE_MATRIX: RETURNS RELATIVE //MATRIX WITHOUT NON_UNIFORM SCALING

Matrix3 Get_Relative_Matrix(INode *node, int t) {

/* Note: This function removes the non-uniform scaling from MAX node transformations. before multiplying the current node by the inverse of its parent. The removal must be done on both nodes before the multiplication and Inverse are applied. This is especially useful for Biped export (which uses non-uniform scaling on its body parts.) */

INode *p_node = node->GetParentNode();

Matrix3 orig_cur_mat; // for current and parent Matrix3 orig_par_mat; // original matrices
Matrix3 cur_mat; // for current and parent Matrix3 par_mat; // decomposed matrices

//Get transformation matrices

orig_cur_mat = node->GetNodeTM(t); orig_par_mat = p_node->GetNodeTM(t);

//Decompose each matrix

cur_mat = Uniform_Matrix(orig_cur_mat); par_mat = Uniform_Matrix(orig_par_mat);

//then return relative matrix in coordinate space of parent return(cur_mat * Inverse( par_mat));

}
