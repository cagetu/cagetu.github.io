--- layout: post title: "look around" date: 2013-03-18 20:03:27 +0900 categories: [개발] --- 열심히 그래픽스 공부와 dx11 공부를 해야하는데, 최근에 웹 생태계와 웹 서비스들이 엄청나서 수많은 개발자들이 저렇게 놀라운 변화를 만들어 낼 수 있을까? 라는 궁금증에 웹 관련한 공부를 해보기로 했습니다.

전 c++ 밖에 모르고, 서버와 데이터베이스에 대한 지식도 거의 없기 때문에 아주 간단한 것부터 시작해보기로 했습니다. ㅎㅎ

php 책을 사고, javascript 책도 사고, 겸사 겸사 python 책도 샀지요... 물론, 류광님이 번역해주신 "Practice Rendering and Computation with Direct3D 11"도 구매했지요. (원서로 보다가 너무 힘들어서 번역서를 샀습니다. 테셀레이션 영문으로 보다가 힘들어서 혼났어요.. 하악~ 하악~ 류광님 오래사세요~ ㅎㅎ)

한번에 너무 급하게 보려고 하니, 좀 부담스럽기는 한데, 그냥 쉬엄 쉬엄 보려고 합니다. ㅎㅎ;;;

웹서비스를 만들어보겠다는 건 아니고, 요즘에 Webkit을 이용해서 Editor 개발에 활용하는 사례들이 많아지고 있어서, 그 부분에 흥미가 생기네요. New Generation of @imsomniacgames tools as webapp와 같은 형태로요...

최근 최적화와 디버깅을 작업을 많이 하게 되었는데, 이 부분에 너무 많은 시간과 노력 + 근성이 요구되어서, 돌이켜보니까 최대한 어떻게든 시각화를 할 수 만 있으면 정말 좋겠다! 라는 생각이 들었어요. 특히, 모바일 환경의 경우, 기기 테스트를 하기 때문에 더욱더 그러하구요. 그래서 web 기반으로 모든 정보를 출력할 수 있다면 정말 좋지 않을까? 라는 생각이 들더라구요...

게임 쪽에 약간 기술적인 이야기가 오고가지 않는데, 요즘에는 게임쪽에서도 웹 기반 기술들을 이용한 이슈들은 지속적으로 이어지는 것 같네요... 그래서 골방 늙은이가 되지 않기 위해서이기도 하고, 겸사 겸사 웹 세상도 구경해보고 싶어졌습니다. 

언넝 directx11으로 멋진 것도 만들고 싶은데, 이거 참.. ㅎㅎㅎ
