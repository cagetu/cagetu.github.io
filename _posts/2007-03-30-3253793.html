But~ d3d msdn 을 봤더니, "D3DSAMP_MIPMAPLODBIAS 기본값은 0 이다" 라고 나오고 끝.. 젠장할 것들.. -_-;;
네이버 파도를 타다 Get... 포스팅 하신 분이 효자 코드라 하시더라.. ㅎㅎ

출처

// 텍스쳐 LOD(Mipmap Detail 설정)
float fLODBias = -3.0f;
pDevice->SetSamplerState( 0, D3DSAMP_MAXMIPLEVEL, 0 );
pDevice->SetSamplerState( 0, D3DSAMP_MIPMAPLODBIAS, *((LPDWORD)(&fLODBias)) );
