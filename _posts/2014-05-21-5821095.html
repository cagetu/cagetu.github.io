* NDK 설치 및 설정

1. Android NDK 다운로드 > 원하는 경로에 압축 풀기
2. 환경 변수 설정
   - ANDROID_NDK 라는 이름으로, 압축 푼 root 폴더를 지정해주자.
   - PATH 에 d:\android-ndk-r8b;d:\android-sdk-windows\platform-tools 추가
   - 편의를 위해 설치 디렉토리를 심볼릭 링크 설정
     (예, ln -s /cygdrive/d/ANDROID/android-ndk-r7c android_ndk)
3. Cygwin 다운로드 및 설치
   -devel 항목과 make 항목을 설치하자
4. Cygwin을 실행시켜서 ndk-builid 를 실행했을 때, 다음과 같이 나오면 성공

5. 패키지에 NDK 코드 추가
  - jni 폴더 추가
  - 제작한 C/mk 파일 추가

6. .mk 파일 (make 파일)
  - android.mk
  - application.mk

7. Native 코드 빌드
  - cygwin 터미널을 실행 시킨 후 JNI 디렉토리로 이동
  - ndk-build 실행

: 이렇게 나오면 성공! 여기까지해서 .so 파일이 나와야 한다.)

주의) NDK에서 wchar_t는 4바이트!!!
일반적으로 Window에서 개발을 할 때, wchar는 2바이트를 사용하기 때문에, 2바이트로 맞춰주고 싶다면 다음과 같이 하면 된다. (참고, NDK에서 wchar_t는 4바이트)

"리눅스에서 wchar_t형을 2byte로 사용하고 싶다면, android-ndk-r6c\toolchains\arm-linux-androideabi-4.4.3 에 보면 setup.mk 파일이 있음. 그 파일을 열어 다음과 같이 추가한다. -fshort-wchar 한 줄 추가하고 컴파일하면 된다.

_TARGET_CFLAGS := \
     -fpic \
     -fstack-protector \
     -fshort-wchar \
     ...
