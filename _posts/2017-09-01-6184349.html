추가해야 할 것도 많지만, 명확하게 무엇을 어떻게 처리해야할지도 몰라서 고민하다가 하나씩 차근 차근 보면서 좀 작은 단위로 구분해보자는 생각이 들었다. 가만히 보니까, 기능 단위로 구분이 가능할 것도 같아보였다. 예를 들어서, RasterizatioinState는 내부적으로 CullMode 설정 FillMode 설정 등으로 구성되어 있다. 기능 단위로 구분을 하다보니, OpenGL 등에서 사용하는 glFillMode, glPolygonMode 같은 기능 함수 형태로 비슷하게 구성이 가능할 것 같았다.

void SetViewportRect(int32 x, int32 y, int32 width, int32 height, float minDepth = 0.0f, float maxDepth = 1.0f);

그래서 이전에 RenderState를 설정해주던 것과 비슷한 느낌으로 함수들을 만들고, 이 함수들을 기반으로 RenderState 변경이 최소화하도록 한번 더 처리를 하려고 생각 중이다. PipelineObject 생성에 대해서는 조금 더 깊이 있게 공부를 해야 할 것 같다. UE4를 참고해서 따라하기도 하는데, 필요 이상으로 복잡하긴 해서 다른 샘플이나 문서를 보면서 연구를 더 해볼 생각이다.
