벡터의 외적은 크게 "두 벡터의 방향성을 판단"하거나 "두 벡터의 교차 여부를 판별"할 때, "Convex Hull"을 만들 때 사용하게 된다. 생각해보면, 특별히 교차 판정을 하는 용도로 사용하기 보다는 두 벡터가 같은 방향으로 보는지를 판정(CCW, CW 등)하거나, 노말 벡터를 구할 때 직각은 벡터를 구하기 위해서만 주로 사용했던 것 같다. Convex를 자주 다루지는 않아서.. ;;;

벡터의 외적의 가장 큰 특징은 방향과 크기를 동시에 가진다는 것이다.

외적을 이용해서 벡터 간의 교차점을 구하면 다음과 같이 되는데

// - 점 a, b를 지나는 직선과 점 c, d를 지나는 직선의 교점을 x에 반환한다.
// - 두 직선이 평행이면(겹치는 경우 포함) 거짓을, 아니면 참을 반환한다.
bool lineIntersection(vector2 a, vector2 b, vector2 c, vector2 d, vector2& x)

{
  double det = (b-a).cross(d-c);
  //두선이 평행인 경우
  if(fabs(det) < EPSILON) return false;
  x = a+(b-a)*((c-a).cross(d-c)/det);
  return true;
}
