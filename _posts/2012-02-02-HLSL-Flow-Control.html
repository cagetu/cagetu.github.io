HLSL supports the following flow-control statements.

  • break
  • continue
  • discard
  • do
  • for
  • if
  • switch
  • while
가장 많이 사용하는 것은 분기문에서의  [branch], [flatten]과 루프에서의 [unroll] 인 듯 함.
  • [branch]는 오직 주어진 조건에 의존하여 if 구문의 한쪽만을 평가함
  • [flatten]은 if 구문의 양쪽을 모두 평가하고 두 결과 사이에서 선택
적절히 활용하자!!!

[unroll(x)] 은 실행이 멈출 때까지 루프를 unroll 한다. 옵션으로 loop가 실행하는 최대 수를 지정할 수 있다.
