--- layout: post title: "[DirectX11] 버퍼 인덱스 설정" date: 2014-10-21 19:04:03 +0900 --- DirectX9에서도 사실 이 부분이 헷갈렸기는 마찬가지이기는 하지만, DirectX11로 넘어오면서 설정하는 부분이 더 많아져서 많이 헷갈렸다. 그 이유는 아래를 보자.

- IASetVertexBuffers(ch, 1, VtxBuffer[], stride[], offsets[])
- IASetIndexBuffer(IdxBuffer, format, offset);
- DrawIndexed(IndexCount, StartIndexLocation, BaseVertexLocation)

예를 Batching을 위해 계속해서 버텍스와 인덱스를 채워 간다고 하면, 인덱스를 어떻게 설정해야 할지가 사실 고민이 많았다.

"Introduction to Buffers in Direct3D 11"를 보니 다음과 같이 계산되는 것을 알 수 있다.


- 버텍스 버퍼 인덱스

- 인덱스 버퍼 인덱스
Start of Index Buffer = Index Buffer Base Address + Offset (bytes) + StartIndexLocation * ElementSize (bytes);

테스트해본 결과로 DrawIndexed에 있는 StartIndexLocation과 BaseVertexLocation은 건드리지 않고, VertexBuffer / IndexBuffer 설정할 때, Vertex Offset과 Index Offset으로만 조절해서 사용하는 것으로 처리해보기로 했다.

만약, 버텍스 버퍼/인덱스 버퍼를 한번에 설정하고, 구간 별로 끊어서 여러 번 렌더링하는 경우라면, DrawIndexed에서 StartIndexLocation이랑 BaseVertexLocation을 조절해서 렌더링할 수 있을 것 같다.
