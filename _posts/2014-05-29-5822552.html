--- layout: post title: "이클립스 프로젝트 생성 및 설정" date: 2014-05-29 14:20:26 +0900 categories: [개발] --- 안드로이드 빌드 및 환경 구축하기에 이이서 안드로이드 프로젝트 설정 및 작업하는 방법에 대해서 정리해보자! 
(이건 개인적인 기록이라서 다른 분들이 보기에 이거 다르게 생각될 수 있습니다!)

1. 이클립스에서 안드로이드 프로젝트 생성
2. Properties에서 Android 항목의 Project Build Target 설정

3. Java Build Path 설정

3-1. 프로젝트의 gen 폴더와 src 폴더를 추가해주고, 빌드 출력 폴더를 설정해준다.

3-2. 라이브러리 추가 및 패스 순서 및 export 설정

4. Java Compiler에서 원하는 자바 컴파일러 버전 설정 ("Compiler compilance level")
5. AndroidManifest.xml 설정
(패키지 이름, 프로그램 버전 정보 설정 등을 설정해준다. 그 밖에 네트워크, 윈도우 형태 등등을 설정해주는데, 이건 나중에 필요할 때 정리를 다시 하도록 해보자!)

이제 안드로이드 빌드를 통해서 만들어진 네이티브 코드 라이브러리를 JNI를 이용해서 연동 시킨 후에, 빌드해서 사용하면 끝!!!
(JNI쪽도 추후 다시 필요할 때 정리해보도록 하겠음.. 일단은 패스~)
