Vulkan API 공부를 시작합니다. 지금 생각해보면 제가 완전한 엔진을 개발해서 게임을 만드는 것까지는 하지 못할 것 같네요. 하지만, API를 기반으로 렌더러는 만들어볼 수 있어서, VulkanAPI 공부를 시작했습니다.

요즘 같이 언리얼 코드를 직접 볼 수 있는 시대에 굳이 API를 만져보는 것이 무슨 의미가 있을까 싶지만, 그래도 직접 만들어보고 그 특성을 파악해보는 것 말고는 이해하고 사용했다고 말할 수 없기 때문에 구현을 해봐야 한다. 정확히는 기기 테스트까지 해봐야 하는데, 그건 어느 정도 그려지면 해보기로 하자..

진짜 잘 사용하려면 멀티 쓰레드 렌더링까지도 봐야 하는데, 일단 api 사용법 중심으로 파악하고 나서 조금씩 확장해 나가보자!
